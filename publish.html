Tools

Here are some tools that we recommend. If you have questions about these or other tools please contact us.

Website

Register your own web domain and create a professional profile at any stage of your academic career. OU Create is a a great place to start.

Create a Blog

Blogging is a good platform for testing research ideas, raising awareness about issues, or developing an online community. Consider aggregating other blog posts with yours to keep it fresh and updated.

OU Create

OU Create offers students, faculty, and staff the opportunity to register a domain (.com, .org, .net) or subdomain and create a digital presence through various mediums such as blogs, portfolios, and wikis.

Wordpress

Wordpress is an open source content management system and blogging tool that makes creating websites easier. It offers a range of design templates to create a website.

Wordpress

Wordpress is an open source content management system and blogging tool that makes creating websites easier. It offers a range of design templates to create a website.

Blogger

Blog-publishing service hosted by Google that allows multi-user blogs with time-stamped entries.

Github

Github is a Git repository hosting service to share open source code. While it is a command line tool, it also offers a web-based user interface to encourage collaboration regardless of skill level.

Drupal

Drupal is an open source software to build websites and databases. It is based on an content management framework and is more robust than Wordpress.

Featured project

OU student Emily Greer developed a blog about her study abroad experience. She used OU Create and Wordpress to create the blog.
