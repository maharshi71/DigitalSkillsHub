Tools

Here are some tools that we recommend to learn the basics of programming. If you have questions about these or other tools please contact us.

Python

A general-purpose, open-source programming language that places an emphasis upon code readability.

Python: Beautiful Soup

A library, written in the Python programming language, for pulling specific pieces of data out of HTML and XML files.

Python: Pandas

An open source, BSD-licensed library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language.

R

Another open source programming language. R was originally developed for statistics but also works well for graphics and text mining.

Featured Project

The GPO Congressional Hearings Search Engine is capable of mining the full text of more than 21,000 hearings (1997-present) of Senate, House, joint or special committee meetings of Congress. Digital tools used for this project were Python Beautiful Soup, Elasticsearch, and Bootstrap.
