Analyze

In a data driven society, understanding the basic concepts, methods, and tools for analyzing information is key. Analysis may involve detecting patterns and correlations within a large textual corpus, geographical data, organizational networks, or scientific data.

  • Tools

Tools

Here are some tools that we recommend for analyzing data. If you have questions about these or other tools please contact us.

Text Analysis

for examining textual information such as literary texts, correspondence, interviews, social media communication, etc.

Spatial Analysis

Uses geographic data with a range of variables to create a spatial model.

Open Refine

A Tool for cleaning and exploring data.

Leaflet

A JavaScript library can be used to create mobile-friendly interactive maps.

Voyant

An easy to use, web-based text analysis software.

StoryMaps

ArcGIS StoryMaps allows users to combine maps with a narrative or multimedia content.

Lexos

Web-based text mining software that leverages computational techniques and statistical analysis to answer literary questions.

GoogleMyMaps

With a Google account, MyMaps allows users to quickly create and publish maps online.

R

Open source programming language. R was originally developed for statistics but also works well for graphics and text mining.

ArcGIS

A suite of software that lets you create, edit, and analyze geographic knowledge. ArcGIS Online enables the creation and sharing data and maps online.

Python

A general-purpose, open-source programming language used for natural language processing. Python has a natural language toolkit for processing the text.

QGIS

An Open Source Geographic Information System (GIS) maintained by the Open Source Geospatial Foundation (OSGeo).

Network Analysis

Analyze data to reveal relationships between entities.

Statistical Analysis

Scientific method for examining large amounts of data to discover patterns and trends.

R

Open source programming language. R was originally developed for statistics but also works well for graphics and text mining.

Excel

Spreadsheet software with calculation, graphing tools, and pivot table options for analyzing data.

D3.js

Javascript library to create interactive data visualizations to study networks and relationships.

R

Open source programming language. R was originally developed for statistics but also works well for graphics and text mining.

SPSS

Statistical software for the analysis of social science data.

SAS

Programming language used for statistical and mathematical analysis.
